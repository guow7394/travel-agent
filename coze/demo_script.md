# Hackathon Demo Script

## Demo 1: 首次规划

用户：

```text
我们两个人明天在上海玩一天，预算500，从人民广场出发也回人民广场。
想去上海博物馆、豫园、外滩、武康路。
A喜欢拍照和美食，但不想走太多路；B喜欢博物馆和高性价比。
帮我规划顺路的一天。
```

预期展示：

- Agent 识别多人偏好冲突。
- 调用 `planCityDay`。
- 生成行动卡、预算账本、交通距离时间、餐厅推荐、静态地图。

## Demo 2: 低预算模式

用户：

```text
预算改成160，尽量省钱。
```

预期展示：

- Agent 切换 `low_budget`。
- 餐厅切到低人均选项。
- 预算风险提示变为 tight/over。

## Demo 3: PlanB

用户：

```text
我们已经逛完上海博物馆了，突然下雨，而且有点累，但外滩还想保留。
```

预期展示：

- Agent 调用 `planB`。
- 保留已完成的上海博物馆。
- 后续切到低体力/雨天策略。

## Demo 4: 复盘更新画像

用户：

```text
今天结束了。外滩夜景很喜欢，但下午走太多路，最后花了420。
下次帮我少安排步行。
```

预期展示：

- Agent 调用 `tripReview`。
- 更新画像为偏低体力、喜欢夜景、保留机动预算。
